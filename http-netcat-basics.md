# Day 4 – HTTP Server & Netcat Basics  

---

## 1. تشغيل خادم HTTP مصغّر (Python)

```bash
python3 -m http.server 8000      # يبدأ خادمًا على المنفذ 8000
```

---

## 2. فحص الشبكة باستخدام Nmap

| الغرض                     | الأمر                                          |
|---------------------------|------------------------------------------------|
| فحص المنافذ المفتوحة      | `nmap 192.168.100.8`                           |
| كشف نسخ الخدمات           | `nmap -sV 192.168.100.8`                       |
| فحص هاتف في الشبكة        | `nmap <phone-ip>`                              |

---

## 3. تنزيل ملفات (wget / curl)

| أداة  | مثال عملي                                              | ماذا تفعل؟                   |
|-------|--------------------------------------------------------|------------------------------|
| wget  | `wget http://192.168.100.8:8000/secret.txt`            | يحفظ الملف في المسار الحالي |
| curl  | `curl -O http://192.168.100.8:8000/secret.txt`         | ‎`-O` يحفظ بالاسم الأصلي    |

---

## 4. Banner Grabbing (Netcat)

```bash
nc 192.168.100.8 8000

GET / HTTP/1.1
Host: test
```

> **Banner Grabbing** = جمع معلومات الخدمة (Server Header، الإصدار)

---

## 5. اكتشاف الأجهزة بالشبكة

```bash
sudo arp-scan --localnet
```

---

## 6. رموز حالة HTTP الشائعة

| الكود | المعنى (EN)   | المعنى (AR)        |
|-------|---------------|--------------------|
| 200   | OK            | الطلب ناجح         |
| 400   | Bad Request   | طلب غير صالح       |

---

## 7. ملاحظات سريعة

| المعلومة                | التفسير                                          |
|--------------------------|--------------------------------------------------|
| **localhost**           | جهازك أنت                                        |
| **banner grabbing**     | جمع بيانات تعريف الخدمة                          |
| **wget / curl**         | تنزيل الملفات / إرسال طلبات HTTP                |
| **nc**                  | اتصال خام لاختبار الخدمات أو نقل الملفات        |
| المنفذ المفتوح = خدمة   | المنفذ مغلق = خدمة متوقفة                        |

---

## 8. تمرين تطبيقي

1. شغّل الخادم بـ `python3 -m http.server 8000`.  
2. استخدم `nmap -p 8000 -sV 192.168.100.8` لمشاهدة الـBanner.  
3. جرِّب `nc 192.168.100.8 8000` واقرأ رأس الاستجابة يدويًّا.  
4. نزّل ملفًا من الخادم باستعمال `wget` ثم `curl` وقارن النتائج.
