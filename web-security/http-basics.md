# Day 5 – HTTP Basics & Status Codes  

---

## 1. مقدّمة سريعة عن HTTP

| عنصر | وصف مختصر |
|------|-----------|
| **URL** | المعرّف الموحّد للمورد |
| **Method** | ‎GET, POST, PUT … |
| **Status Code** | نتيجة الطلب (200, 404…) |

---

## 2. رموز الحالة الشائعة

| الكود | المعنى (EN) | المعنى (AR) |
|-------|-------------|-------------|
| **200** | OK | نجح الطلب |
| **301** | Moved Permanently | نقل دائم |
| **403** | Forbidden | ممنوع |
| **404** | Not Found | غير موجود |
| **500** | Internal Server Error | خطأ داخلي |
