## Linux Commands

| أمر          | الوصف المختصر      |
|--------------|--------------------|
| `ls`         | عرض الملفات        |
| `cd`         | تغيير المجلّد      |
| `pwd`        | المسار الحالي      |
| `cat`        | قراءة ملف          |
| `grep`       | البحث داخل نصّ     |
| `find`       | البحث عن ملفات     |
| `chmod`      | تغيير الأذونات     |
| `chown`      | تغيير المالك       |
| `history`    | عرض الأوامر السابقة |

```bash
# أمثلة
ls -la
grep "root" /etc/passwd
find / -name "*.conf"
