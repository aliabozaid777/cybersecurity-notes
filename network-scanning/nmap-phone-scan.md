# Day 11 – Nmap Scan on Mobile Device  

---

## 1. تحديد IP الهاتف

```bash
# على كالي
sudo nmap -sn 192.168.100.0/24
```

ابحث عن MAC Vendor يشير إلى **Samsung / Apple** أو اسم الجهاز.

---

## 2. فحص المنافذ الشائعة

```bash
nmap -sS -Pn -p 1-1000 <phone-ip>
```

| خيار | تفسير |
|------|-------|
| `-sS` | فحص SYN تخفيّ |
| `-Pn` | تخطّي اكتشاف الـPing |
| `-p 1-1000` | أول ألف منفذ |

---

## 3. التحقق من الخدمات

إذا ظهر منفذ **5555/tcp open adb** ⇒ Android Debug Bridge مفعَّل (خطر أمني).

---

## 4. توصيات

- تعطيل ADB عبر USB فقط، لا عبر TCP.  
- استخدام جدار حماية للهاتف (NetGuard).
