# Day 13 – Local Assessment: Ports 8080 & 9090  

---

## 1. فحص محدد

```bash
nmap -sV -p 8080,9090 127.0.0.1
```

| منفذ | مفسَّر عادةً كـ | ملاحظات |
|------|----------------|---------|
| **8080** | HTTP بديل    | قد يكون لوحة تحكم WebApp |
| **9090** | Proxy/WS     | أحيانًا لوحة إدارة مساحات العمل |

---

## 2. اختبار Banner يدوي

```bash
nc 127.0.0.1 8080
HEAD / HTTP/1.0
```

---

## 3. توصيات

- أوقف الخدمات غير الضرورية.  
- أضف كلمة مرور للوحة التحكم أو قيّد بالـIP.
